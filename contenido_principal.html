O.V.I Diseño de sitios web

Escrito el 28/03/2018 a las 11:30

Estimado (a) estudiante, sea bienvenido al Objeto Virtual de Información (O.V.I) diseñado por los estudiantes del curso de Diseño de Sitios Web, que contiene información sobre las temáticas generales del curso. En este material encontrará contenidos de lectura y multimediales que le ayudaran en el desarrollo de las actividades de aprendizaje.
Es importante que navegue por las diferentes secciones y revise todo el material disponible, con esto lograra adquirir y afianzar sus conocimientos. Se ha tratado de consolidar para este O.V.I. un banco de materiales actualizado y con estrecha relación con los contenidos y las actividades del curso.
Cualquier inquietud frente a este O.V.I. se estará pendiente para brindarle la ayuda y soporte necesarios, muchos éxitos en sus actividades académicas y se le desea que con ayuda de este material las pueda desarrollar de la mejor forma.


Objetivos

- Aprender a utilizar los comandos de GIT commit, push, pull, fetch a crear ramas branch y merge, así como utilizar GIT a través de una interfaz gráfica en este caso GitHub Desktop.
- Crear un repositorio y sincronizarlo con GitHub, así como clonar repositorios desde GitHub a la computadora local para poder trabajar con ellos.
- Conocer los diferentes editores de código que podemos utilizar para nuestros desarrollos como Sublime text, notepad ++, brackets y otros editores online como CodePen asi como conocer herramientas que nos permitan crear mockups o wireframes para realizar una maquetación previa de nuestro sitio web.
- Aprender la sintaxis y utilización del lenguaje de etiquetado HTML5 y el lenguaje de estilos CSS3, para diseñar sitios web conforme a las necesidades y requerimientos del usuario final.


Contenido

- Respositorios Git y GitHub.
- HTML5.
- CSS3.
- Editores de Codigo.
- Informacion Adicional.
